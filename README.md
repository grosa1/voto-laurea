## voto-laurea

Semplice script per il calcolo della media pesata e la base di laurea.

Utilizzo:
```
py main.py <lista_esami.txt>
```

Esempio di output:
```
1. esame1 | 6 | 30

2. esame2 | 8 | 28

3. esame3 | 3 | 25

4. esame4 | 10 | 18

5. esame5 | 9 | 20

6. esame6 | 12 | 27

7. esame7 | 3 | 26

#############################################

media: 24.33

base di laurea: 89.0
```



Il file lista esami, indipendentemente dal formato, deve essere strutturato cos√¨:
```
<nome esame>,<peso in crediti>,<voto>
...
esame1,6,30
esame2,8,28
esame3,3,25
...
```

P.S. le lodi non contano, quindi in caso di 30 e lode inserire semplicemente 30
